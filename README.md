# modcv - Tkinter Video Player
A lightweight video player with out sound and editor built with Python, Tkinter, and OpenCV.

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![OpenCV](https://img.shields.io/badge/OpenCV-4.8%2B-orange)

## Features

-  Play video files (MP4, AVI, MKV)
-  Take snapshots with automatic naming
-  Create GIFs from video frames
-  Pause/resume playback
-  Seek through video with slider
-  Switch between videos without restarting
